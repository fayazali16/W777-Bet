W777 Bet Information Hub — Next.js (App Router)
==============================================

app/                   Routes, layouts, globals.css, robots.ts, sitemap.ts, manifest.ts
app/about/             /about/
app/contact/           /contact/ + ContactForm (server action)
app/disclaimer/        /disclaimer/
app/privacy-policy/    /privacy-policy/
app/blog/page.tsx      /blog/
app/blog/[slug]/       /blog/download/, /withdraw/, /deposit/, /signup/
app/blog/category/[slug]/  e.g. /blog/category/account-payments/
app/guide/page.tsx     /guide/
app/faq/               /faq/ + FaqAccordion (client)
app/responsible-gambling/page.tsx
lib/blog/              categories + posts data, queries, SEO fields
components/            SiteHeader (client), SiteFooter, JsonLd
lib/                   site.ts, jsonld.ts, faq-data.ts
public/
  assets/              icon + og image (served at /assets/...)
  uploads/images/      Optional user uploads

Commands: npm run dev | npm run build | npm run start
  If you see "Cannot find module './NNN.js'": npm run clean && npm run dev
  Dev mode disables webpack filesystem cache (next.config.ts) to reduce stale chunks.
  Use Next.js only (port 3000) for this app—not python -m http.server.
Env: copy .env.example → .env.local and set NEXT_PUBLIC_SITE_URL
